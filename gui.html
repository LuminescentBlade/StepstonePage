Graphical User Interface

In-Game Menu - While the game is in progress, we'll always have a game menu that allows the player to select from the following options:

  • New Game - If clicked while game is in progress, it will ask the user if they really want to start a new game and exit the current game. If yes, or if no current game in progress, start new game.
  • Pause Game - If clicked while game is in progress, it will be the same thing as pressing ESC. Otherwise it won't be active.
  • Display Controls - If clicked, it will display info about the controls.
  • About - If clicked, it will display info about the creators.
  • Exit - If clicked, the game will exit.
